# Placeholder audit

Use this table to track placeholder replacement during setup. The first agent
owns this document and should update status as placeholders are resolved.

## How to use

- Add an entry for each file containing `{{...}}` placeholders.
- List all placeholders in the file.
- Set owner to the agent responsible for updates.
- Track status as Pending / In Progress / Done.
